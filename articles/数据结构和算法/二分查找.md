# 二分查找


## 介绍

想象你是一个老师，在一堆试卷中，你想要找出某张试卷，你会怎么去做，才能在最短的时间里找到它。


## 抽象

```js
    function bsearch(A, x)

    A：数组
    x：需要查找的值
    返回：x 在 A中的位置，不存在返回 -1
```

## 循环不变式

```
    循环前：
        l 表示：查找范围的左边界
        r 表示：查找范围的右边界
        guess 表示：l，r 的中间位置

    循环后：
        l 表示：新的查找范围的左边界
        r 表示：新的查找范围的右边界
        guess 表示：新的l，r中间位置  

    每次循环结束后，查找的值要么在位置 l-r 之间，要么不存在      
```

## 程序实现

```js
    // 在一组已经排序好的数中查找，如果找到那就返回索引 未找到返回 -1
```
